#  World Cup Database Project

This is a relational database project built as part of the FreeCodeCamp Relational Database Certification.

The project uses PostgreSQL, Bash scripting, and SQL to process World Cup match data from a CSV file and perform structured queries on it.

---

##  Project Description

The goal of this project is to:

- Create a PostgreSQL database called `worldcup`
- Design relational tables for teams and games
- Import match data from `games.csv`
- Build a script to insert data efficiently into the database
- Run SQL queries to analyze World Cup statistics
